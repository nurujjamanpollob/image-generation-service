Comprehensive guide to using the Image Generation API

API Endpoints

Text-to-Image Generation

POST /v1/generate/text-to-image

Generate images from text prompts using AI models.

Request Headers:

X-API-KEY: your_secret_api_key

Request Body (JSON):

{
  "prompt": "A cinematic shot of a raccoon astronaut, adrift in space, looking at Earth",
  "num_images": 1,
  "negative_prompt": "blurry, low quality, cartoon",
  "guidance_scale": 7.5
}

Response:

{
  "status": "success",
  "image_urls": [
    "http://localhost:8000/downloads/123456789.png?api_key=your_api_key"
  ],
  "generated_at": "2023-01-01T00:00:00Z"
}

Image-to-Image Generation

POST /v1/generate/image-to-image

Modify existing images using text prompts.

Request Headers:

X-API-KEY: your_secret_api_key

Request Body (multipart/form-data):

  • reference_image: The image file to be modified
  • prompt: A text string describing the desired modifications
  • num_images (optional): Number of variations to generate
  • strength (optional): How much to transform the original image (0-1)

Response:

{
  "status": "success",
  "image_urls": [
    "http://localhost:8000/downloads/123456789.png?api_key=your_api_key"
  ],
  "generated_at": "2023-01-01T00:00:00Z"
}

Image Upscaling

POST /v1/upscale/image

Enhance image resolution while preserving quality.

Request Headers:

X-API-KEY: your_secret_api_key

Request Body (multipart/form-data):

  • image_to_upscale: The low-resolution image file
  • prompt(optional): Prompt to upscale the image.

Response:

{
  "status": "success",
  "image_url": "http://localhost:8000/downloads/123456789.png?api_key=your_api_key",
  "original_resolution": "512x512",
  "new_resolution": "1024x1024",
  "generated_at": "2023-01-01T00:00:00Z"
}

Secure Image Download

GET /downloads/{filename}?api_key={api_key}

Download generated images with embedded API keys for secure access.

Admin Endpoints

These endpoints require the ADMIN_API_KEY to be set in environment variables.

Create New API Key

POST /admin/keys

Generate a new API key for users to access the service.

Request Headers:

X-API-KEY: your_admin_api_key

Response:

{
  "message": "API key created successfully",
  "api_key": "generated_api_key_here"
}

Revoke API Key

DELETE /admin/keys/{api_key}

Revoke (delete) an existing API key.

Request Headers:

X-API-KEY: your_admin_api_key

Response:

{
  "message": "API key revoked successfully"
}

List API Keys

GET /admin/keys

List all active API keys.

Request Headers:

X-API-KEY: your_admin_api_key

Response:

{
  "api_keys": [
    "key1",
    "key2",
    "key3"
  ]
}
